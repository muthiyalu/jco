JCloudObjects is a opiniated DRY based Micro Services framework based on the Spring Suite

JCloudObjects can make coding in Java fun again!

JCloudObjects leverages the power of the Spring suit and adds key automation and Data Lifecycle management features to accelerate Java based Application development



Across a layered Archiecture



Channel layer- Browser , Mobile and Edge features



Channel Gateway- Gateway services, Throttling, Auth Microservices Proxy , Discovery etc



Components Layer - Rich set of configurable Microservices for common functions like Secure Vault Config, reporting , logging etc



Rich domain model that uses patterns like Property Groups, Logical Data Models, Remoting to abstract and simplify data access and lifecycle management



Infra as Code-Once clock deploy scripts to Cloud Platforms through Containers-Kubernetes, Pivotal Cloud Foundry and Rancher



Out of the Box CI/CD integration based on Jenkins Blue Ocean



UI based Configuration of layers

The main mission if JCloudObjects is to abstract the complexities of Java based Cloud Native Applications without Sacrificing control

Self Aware Data layer(upcoming feature)

Data layer that moves beyond simple ORM and enables modelling of complex cloud native cognitive applications !

Enterprise Grade Security

Based practice Security model built on Spring Boot 2.0 and Spring Security 5.0
